# FASE 08: Sistema de Categorías

## 🎯 Objetivo

CRUD de categorías personalizadas (las predefinidas no se pueden editar/eliminar).

## 📋 Tareas Principales

### 8.1: Página de categorías

- Listar todas (predefinidas + del usuario)
- Distinguir visualmente predefinidas vs personalizadas

### 8.2: Formulario crear categoría

- Nombre, tipo (income/expense), color, icon picker

### 8.3: Editar/Eliminar categorías

- Solo categorías del usuario (user_id not null)
- Validar que no tenga transacciones antes de eliminar

### 8.4: Icon picker component

- Emoji picker o lista de iconos predefinidos

gh pr create --base main --head develop --title "FASE 08: Sistema de Categorías"
